Change the master password used to encrypt and decrypt your vault.

You must enter your current master password to authorize the change.
The new password must meet the security requirements:
- At least 12 characters long
- Contains at least one uppercase letter
- Contains at least one lowercase letter
- Contains at least one digit
- Contains at least one special character or symbol

This operation will re-encrypt your vault with the new password.

Usage:
  pass-cli change-password [flags]

Examples:
  # Change master password
  pass-cli change-password

Flags:
  -h, --help   help for change-password

Global Flags:
      --config string   config file (default is $HOME/.pass-cli/config.yaml)
      --vault string    vault file path (default is $HOME/.pass-cli/vault.enc)
  -v, --verbose         verbose output
